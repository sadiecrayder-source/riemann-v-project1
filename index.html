Advanced Riemann Analysis

1. The Definition of the Definite Integral

Riemann Sums are the formal construction of the integral. For a continuous function \(f\) on \([a, b]\):

  • The Partition: We divide \([a, b]\) into \(n\) subintervals where \(\Delta x = \frac{b-a}{n}\).
  • The Selection: We choose a sample point \(x_i^*\) in each subinterval.
  • The Sum: \(S_n = \sum_{i=1}^n f(x_i^*) \Delta x\).

2. Overestimates vs. Underestimates

Understanding the relationship between the function's behavior and the sum type is vital for the AP Exam:

  • Left Riemann Sum (LRAM): If \(f(x)\) is increasing, LRAM is an underestimate. If decreasing, it is an overestimate.
  • Right Riemann Sum (RRAM): If \(f(x)\) is increasing, RRAM is an overestimate. If decreasing, it is an underestimate.
  • Midpoint Sum (MRAM): Accuracy depends on concavity. For a concave up function, Midpoint typically underestimates the actual area.
